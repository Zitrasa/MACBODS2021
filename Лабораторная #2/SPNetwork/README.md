# SP подстановка

## Входные данные
1. Номер в списке: 19, значит исходный текст: 19 * 7 = 133 (1000 0101)
2. В имени и фамилии шесть букв, значит ключ: 4096 - 11 * 6 * 6 = 3700 (1110 0111 0100)
